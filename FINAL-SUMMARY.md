## 📂 HOW TO USE

### Step 1: Download Files
All 4 optimized HTML files are in `/mnt/user-data/outputs/`:
- index-optimized.html
- comercial-optimized.html
- eventos-optimized.html
- contacto-optimized.html

### Step 2: Rename Files (Remove "-optimized" Suffix)
```
index-optimized.html → index.html
comercial-optimized.html → comercial.html
eventos-optimized.html → eventos.html
contacto-optimized.html → contacto.html
```

### Step 3: Update Contact Information

**Email:** Replace `hola@matirath.jpg`
- Find & replace in: comercial.html, eventos.html, contacto.html
- Search for: `mailto:hola@matirath.jpg`
- Replace with: `mailto:YOUR-EMAIL@DOMAIN.COM`

**WhatsApp:** Replace number in contacto.html
- Find: `https://wa.me/5491122334455`
- Replace with: `https://wa.me/YOUR-COUNTRY-CODE+NUMBER`
- Example: `https://wa.me/5491234567890` (no spaces, no +)

### Step 4: Add Real Images

Replace fuchsia (#FF00FF) placeholders with your photography:

**Option A: Inline CSS**
```html
<div class="gal-img" style="background-image: url('imgs/image1.jpg')">IMAGEN</div>
```

**Option B: External CSS**
```css
.gal-card:nth-child(1) .gal-img {
  background-image: url('imgs/mutuo1.jpg');
  background-size: cover;
  background-position: center;
}
```

### Step 5: Upload to Server

Directory structure:
```
your-domain.com/
├── index.html
├── comercial.html
├── eventos.html
├── contacto.html
└── imgs/
    ├── concrete.webp (texture, keep this)
    ├── og-cover.jpg (social preview, keep this)
    └── [your photos here]
```

## 🔧 CUSTOMIZATION GUIDE

### Change Colors
Edit CSS variables at top of `<style>`:
```css
:root {
  --blk: #0a0908;      /* Change main dark */
  --cream: #e8e1d5;    /* Change light */
  --gold: #b08c18;     /* Change accent */
  --gutter: 3rem;      /* Change spacing */
}
```

### Change Marquee Text (index.html)
Find `.mq-t` section and edit the `<span>` elements with keywords.

### Add New Section
1. Add `<section class="YOUR-SECTION s-dark">...</section>`
2. Add CSS in PAGE-SPECIFIC STYLES section
3. Add reveal class (`.r`) for scroll animations
4. Follow existing grid patterns

### Modify Navigation
Edit `<nav>` section (affects all pages since it's SHARED).
